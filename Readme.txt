# Tempec

Tempec es una unidad de conservacion utilizada para convervar medicamentos, sangre, semen, etc. de distintas especies. en el sector pecuario.

Antecedentes:
El manejo y control de la temperatura en las unidades actuales es inexacto, el rango en el que actua tanto compresor (enfria) y resistencia (calienta) es muy grande, teniendo un minimno de 1 grado centrigado arriba y 1 grado abajo para funcionar

Justificaciòn:
El contar con un rango que permita a los equipos variar la potencia con la que pueden enfriar o calentar la unidad garantiza presiciòn, asi como el hecho de crear una interfaz que muestre en tiempo real y un historial de las condiciones de temperatura.

Objetivo:
Desarrollar el software sobre un microcontrolador esp32 para controlar una unidad de conservaciòn, el cual debe ser operado atravez de una interfaz gràfica basada en una pantalla nextion de 3.2" 

Alcances:
-El dispositivo debe poder controlar 2 salidas de 10 Amp AC cada una
-la interfaz debe estar dotada de un menu de navegacion con funciones como historial de temperatura, establecer setpoint, rango de operacion, hora, etc.
-El historial debe ser de por lo menos 1 mes
-Sensor de temperatura DS18B20
-Sensor de humedad
-Sensor de presencia de energia electrica
